라북 V6 완성본

반영 사항
1. 헤더 메뉴 복원 및 재구성
   - 홈
   - 국가자격증
   - 민간자격증
   - 비교
   - 랭킹
   - 수험가이드
   - 우측 파란색 '자격증 검색' 버튼
2. 모바일 햄버거 메뉴에도 동일 메뉴 적용
3. 모바일 상세 히어로 제목 아래 과도한 공백 축소
   - pb-[290px] -> pb-[160px] / sm:pb-[190px]
4. 모바일 히어로 이미지 위치 조정
   - object-[70%_55%]
5. 기존 상세페이지 구성과 디자인 유지
6. page.tsx JSX 문법 오류 수정
7. Google 폰트 외부 의존성 제거로 빌드 안정화
8. npm run build 검증 완료

실행 방법
npm install
npm run dev
