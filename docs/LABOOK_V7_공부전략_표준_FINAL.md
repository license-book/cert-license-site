# LABOOK V7 공부전략 표준 (FINAL)

- 상태: 최종 확정
- 적용 범위: 국가자격증, 국가공인 민간자격, 등록 민간자격
- 운영 원칙: 공통 컴포넌트 1개 + 자격증별 JSON 데이터
- FAQ: 상세페이지 하단 공통 FAQ를 사용하며 공부전략 내부에는 별도 FAQ를 두지 않음

## 최종 섹션

1. 핵심 전략
2. 필기 공부 전략
3. 실기·작업형·면접·논술·실습 준비 전략
4. 추천 학습 순서
5. 현실적인 공부 기간
6. 시간이 부족하다면 이렇게 공부하세요
7. 합격 팁
8. 많이 실패하는 이유
9. 시험 전 체크리스트
10. 공부 자료 활용법
11. 실제 합격자들이 많이 선택하는 공부 순서
12. 라북 추천 로드맵
13. 라북 한 줄 조언

## 적용 원칙

### 공통 구조
모든 자격증은 동일한 13개 섹션을 사용한다. 사용자는 어느 자격증 페이지에서도 같은 위치에서 필요한 정보를 찾을 수 있어야 한다.

### 시험 유형 자동 분기
`examType` 값에 따라 3번 섹션의 제목을 자동으로 바꾼다.

- `written_practical`: 실기 공부 전략
- `written_performance`: 작업형 준비 전략
- `written_interview`: 면접 준비 전략
- `written_essay`: 논술 준비 전략
- `written_practice`: 실습 준비 전략
- `written_only`: 필기 심화 전략
- `practical_only`: 실기 준비 전략
- `performance_only`: 작업형 준비 전략
- `interview_only`: 면접 준비 전략

### 선택 항목
시험에 해당하지 않는 항목은 빈 배열이나 빈 문자열로 두지 말고 JSON에서 생략할 수 있다. 컴포넌트는 값이 없는 블록을 렌더링하지 않는다.

### 콘텐츠 작성 원칙
- 과장된 합격 보장 표현 금지
- 공식 출제기준과 실제 준비 흐름을 구분
- 수험생이 바로 행동할 수 있는 구체적인 문장 사용
- 단순 이론 나열보다 우선순위, 반복 횟수, 완료 기준 중심
- 민간자격은 발급기관별 시험 차이를 명시
- 취업 효과나 활용도는 자격증 자체 효력과 분리해 설명
- “실제 합격자” 표현은 조사·후기·일반적으로 관찰되는 경향을 근거로 작성하고, 근거가 약하면 “많이 활용되는 방식”으로 완화

## 데이터 품질 체크

- [ ] 13개 섹션이 의도대로 노출되는가
- [ ] 시험 유형에 맞는 제목이 표시되는가
- [ ] 공부 기간이 초보자·평균·경험자로 구분되는가
- [ ] 시간 부족 전략이 1주·2주·1개월 기준으로 작성되었는가
- [ ] 실패 이유마다 예방법이 있는가
- [ ] 체크리스트가 실제 시험 준비물과 일치하는가
- [ ] 자료 활용 순서가 구체적인가
- [ ] 추천 로드맵 내부 링크가 유효한가
- [ ] 민간자격의 기관별 차이 또는 유의사항이 필요한 경우 포함되었는가
- [ ] 페이지 하단 FAQ와 중복되지 않는가
